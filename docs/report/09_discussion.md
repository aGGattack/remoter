## 8.2 Limitations

- **Blocking architecture** - Main loop blocks during transmission
- **No encryption** - Communication is plaintext
- **Single channel** - No frequency hopping
- **Power consumption** - Not optimized for battery life
- **No error correction** - Relies only on ACK

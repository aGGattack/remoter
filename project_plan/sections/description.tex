\subsection*{English}
For outdoor autonomous mobile robots, the current solution relying on a gamepad is too limited. This project aims to develop a long-range wireless remote control system with safety features for autonomous robots. The system must detect loss of communication and cut power to motors, allow manual override of velocity and turn rate, and operate for at least 8 hours on battery. Optionally, the remote should display messages from the robot such as battery status.

The project involves circuit and PCB design, as well as 3D-printed casing for the devices.

\subsection*{Dansk}
For autonome mobile udendørs robotter er den nuværende løsning med en gamepad for begrænset. Dette projekt har til formål at udvikle et langtrækkende trådløst fjernbetjeningssystem med sikkerhedsfunktioner til autonome robotter. Systemet skal detektere tab af kommunikation og slukke for strømmen til motorerne, tillade manuel overstyring af hastighed og svinghastighed og fungere i mindst 8 timer på batteri. Fjernbetjeningen skal eventuelt vise beskeder fra robotten, såsom batteristatus.

Projektet involverer kredsløbs- og PCB-design samt 3D-printet kabinet til enhederne.

The project scope includes the following requirements:

\begin{enumerate}
    \item \textbf{Communication Loss Detection:} System must detect when communication is lost and immediately cut power to the robot's motors.
    
    \item \textbf{Manual Control Override:} The remote must allow the operator to take over control of the vehicle, controlling drive velocity and turn rate.
    
    \item \textbf{Power Supply - Robot Side:} The mobile robot shall power the on-board receiver/control unit.
    
    \item \textbf{Power Supply - Remote Side:} The hand-held remote shall be battery powered with at least 8 hours of operation on a fully charged battery.
    
    \item \textbf{Status Display (Optional):} The remote end should be able to display messages from the robot, e.g., remaining battery power.
    
    \item \textbf{Communication Frequency:} The system shall use the 433 MHz band for communication.
    
    \item \textbf{Hardware Development:} The project shall include circuit and PCB design for both the remote and robot units.
    
    \item \textbf{Enclosure Design:} 3D-printed casing shall be designed and produced for both devices.
\end{enumerate}

\subsection*{Out of Scope}
\begin{itemize}
    \item Development of the autonomous navigation algorithms
    \item Integration with existing robot control systems (beyond motor power control)
    \item Production of multiple units beyond the prototype
    \item Regulatory certification (CE, FCC, etc.)
\end{itemize}

## Overview

This report presents the design and implementation of a wireless remote controller system using Arduino-compatible hardware. The system utilizes the Adafruit Feather M0 microcontroller with an RFM69 LoRa transceiver to enable reliable wireless communication between a handheld remote and a receiving unit.

## Key Contributions

- Design of a low-cost wireless remote controller
- Implementation of custom communication protocol
- Integration of joystick input, OLED display, and radio modules
